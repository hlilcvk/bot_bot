# AI Canonical Mapping

Deterministic backend truth is the source of record.

Canonical AI roles:

- analyzer
- explainer
- risk_critic
- operator_recommender
- auditor_commentary

Canonical AI inputs:

- scores
- realization metrics
- regime state
- health state
- telegram state
- entitlement state
- CRM profile
- finance mirror
- audit findings

Canonical AI outputs:

- structured analysis
- evidence explanation
- risk narrative
- operator recommendation
- audit commentary

Canonical permissions:

- read_truth
- suggest_actions
- summarize_audit

AI does not receive execution authority.
