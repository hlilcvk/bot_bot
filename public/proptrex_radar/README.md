# PROPTREX Radar

Canonical sub-product tree.

## Top-level folders

- `assets/` - runtime CSS, JS and editable config
- `contracts/` - page, navigation and locale registries
- `locale/` - product message catalogs
- `ui/` - product surface sections
- `domain/` - signal, scoring and AI domain groups
- `api/` - endpoint and websocket contracts
- `persistence/` - event, operational, analytics and AI stores
- `workers/` - evaluator and realtime jobs
- `ai/` - AI service and prompt layer
- `deploy/` - deployment scaffolding
- `health/` - health contract surface
- `observability/` - logging and telemetry surface
- `evidence/` - build and smoke proof outputs
- `tests/` - unit, integration and e2e slots
